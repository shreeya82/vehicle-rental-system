# vehicle-rental-system
A basic vehicle rental system project developed in Java/PHP.

**Features:**
- Add and remove vehicles
- Book and return vehicles
- Search vehicles

**Note:** Database functionality is implemented with MySQL.

## Project Structure
- `src/` - Source code files
- `db/` - Database scripts
- `assets/` - CSS, images, and frontend files
- `docs/` - Documentation
- `tests/` - Sample tests and inputs
- 
**How to run:**
1. Import the SQL database file (optional)
2. Open project files in your IDE
3. Run the main program / open index.php in browser
